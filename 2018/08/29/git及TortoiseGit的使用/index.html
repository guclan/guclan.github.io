Windows下git及TortoiseGit的使用

发表于 2018-08-29 | 分类于 工具心得

Git的安装

  • Git官网
  • TortoiseGit官网

Git介绍

Git是一个开源的工具，用来做版本控制，但由于是开源，没有像SVN那样严格的权限管理，所以在企业项目中使用的时候需要谨慎操作或在Git服务器上做权限控制。

TortoiseGit是一个图形化的Git客户端工具包，它可以进行Git绝大多数的操作，大大减少了使用工具的学习成本。

准备阶段

从Git官网下载好安装包，TortoiseGit官网下载安装包以及对应的语言包。共计3个安装包。

安装

安装Git

双击git安装包安装即可

参数如下：

默认勾选了Git GUI Here，也可使用默认选项，Git的GUI界面不是很灵活，建议显示bash就可以

选择Git使用的编辑器，建议使用Vim作为编辑器，这样可以使用git bash 直接调用，一般不会使用此功能，所以无所谓

默认选项即可，这里是指定可以在哪调用git的命令

默认选择OpenSSL即可，这里是选择git通信加密方式

如果你克隆的代码会从本地复制到Linux中建议选择第二个，否则默认即可，建议选择第二个

默认选项即可，这里是默认控制台窗口

默认选项即可，点击install进行安装

安装TortoiseGit

Git工具安装后，要开始安装TortoiseGit

安装过程这里会省略部分基础步骤

这里的选项选择OpenSSH，一定要选择OpenSSH，不然你的密钥无法在TortoiseGit中直接使用，而是需要进行转换

默认选项即可

安装完成时默认勾选了第一次的初始化设置，此时进入设置向导不要进行任何操作

在向导开启状态安装中文pack

安装后发现语言还是只有English，别慌，点击一下refresh

此时已经出现中文选项，点击选择，并点击下一步

Git路径选择你安装的Git程序的bin目录即可，如果Git是默认安装的，这里不用修改

名称和Email建议填写你在此计算机中提交最常用的名字和邮箱，如果是公司的gitlab建议填写gitlab分配的用户名和公司邮箱

SSH选项选择OpenSSH，这里一定要确认选择的是OpenSSH，点击完成即可开始使用

由于git和TortoiseGit是免费的，建议一切安装包从官方网站下载。
